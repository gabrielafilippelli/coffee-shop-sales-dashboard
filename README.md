# Coffee Shop Sales Dashboard

## Project Overview
Analyzed transaction records from a mock coffee shop chain across three NYC locations. Created an interactive Excel dashboard to visualize sales trends, peak hours, product performance, and revenue drivers, supporting data-driven business decisions.

---

## Analytical Questions
- How have sales trended over time across locations?
- Which days and hours are busiest?
- Which products drive the most revenue?
- How can insights inform staffing and inventory management?

---

## Tools & Technologies
- Microsoft Excel
- PivotTables, charts, conditional formatting
- Optional: Power Query for data cleaning

---

## Key Findings
- Identified peak sales hours and busy days across locations
- Highlighted top-selling and revenue-driving products
- Revealed differences in sales trends by store location
- Dashboard supports actionable decisions for staffing and inventory
